# Desarrollo

## Preparación prevista

El proyecto parte de un entorno virtual Python local.

Dependencias principales previstas:

- `PySide6`: interfaz gráfica de escritorio.
- `httpx`: cliente HTTP.
- `beautifulsoup4`: análisis de HTML.
- `pytest`: pruebas.
- `ruff`: formato y comprobaciones básicas.

## Primeros hitos

1. Configuración local no sensible.
2. Cliente HTTP con login contra Moodle.
3. Parser de cursos desde HTML.
4. Validación de acceso a cursos por id.
5. Interfaz mínima con formulario de conexión.
6. Tabla/listado de cursos.
7. Barra de progreso y mensajes de error.

## Convenciones iniciales

- Código y documentación en español.
- Commits cuando haya un estado comprobado.
- Sin ramas salvo que el desarrollo lo pida más adelante.
- La contraseña no se almacena.
- Los datos de alumnado no se guardan internamente.
